Speed control
  • When you hold down an arrow key, Image Viewer will stay at the first/last image before going to other end.
    When the delay ends, your action continues unless it’s interrupted.
    Set debounce period to 0 if you don't want this behavior.
  • When you hold down an arrow key, Image Viewer cycles through images at approximately the specified interval.
    The default setting 80ms = 12.5fps (12.5 image per 1 second).
    When the throttle period is set near 0, the maximum frame rate is limited only by your computer's performance.
  • When you use auto navigation, Image viewer will navigate to the next image approximate this period of time.
    When auto period was set to be close to 0, fps limit will only depending on you computer performance.
  • When the cursor is over the control panel, the Image Viewer will hide the control panel after the specified time.
    Set the panel timeout to 0 to disable this feature.
Image search hotkey

  • When more than one search using same hotkey, all search will be triggered, built-in hotkey like 'Ctrl + W' or 'Ctrl + T' will not be overridden.

    When using custom search, '{imgSrc}' will be used to replace with real URL

    Soft disable hotkey without Ctrl/Alt/Shift key, import a modified settings to save it.

Domain specific settings


  • Image Viewer allows you to select images by right-clicking on them.
    By default, it removes hover elements to select video thumbnails.
    Add domains here if you want to keep the hover elements for those websites.



  • Some web pages load more content as you scroll down.
    Add domains here to enable active auto scroll for those websites.



  • Some web pages serve extremely large images through lazy loading (eg. Unsplash).
    Add domains here if you want to disable image unlazy for those websites.



  • Some SPA (Single Page Application) don't change URL when switching between pages.
    Add domains here if you only want fresh images.

  • Example setting (support regex):
    example.com
    sub.example.com
    another.com
    /.+\.regex\.com\/path/
